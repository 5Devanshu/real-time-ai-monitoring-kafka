# Real-Time AI Monitoring Pipeline with Kafka

This project implements a real-time monitoring pipeline using Kafka and Python for processing logs, alerts, and system events with sub-second latency.

## Features:
- Implemented Producers/Consumers, schema validation, and anomaly detection routing.
- Added AI-based event classification using lightweight REST microservices.
- Developed a Streamlit dashboard for real-time system health and alert tracking.

## Technologies Used:
- Python
- Apache Kafka
- Streamlit
- REST Microservices
